# Should You Upgrade?

A simple web app that helps you decide whether itâ€™s worth upgrading your tech device.

## What it does
- Asks about your device type, age, satisfaction, problems, and budget
- Gives a verdict: **Upgrade Now**, **Wait**, or **Not Worth It**
- Suggests cheaper alternatives when possible (battery, RAM, storage, etc.)

## Why
To avoid unnecessary upgrades and save money by making smarter tech decisions.

## How to use
1. Download or clone the project
2. Open `index.html` in your browser
3. Adjust the sliders and options
4. Click **Check Now**
